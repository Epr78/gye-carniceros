# GyE Carniceros

Proyecto DAW de una web de carnicería desarrollado con:

- PHP
- MySQL
- Apache
- HTML
- CSS
- JavaScript

## Entorno local

Ruta del proyecto:

C:\xampp\htdocs\gye-carniceros

Dominio local:

http://gye-carniceros.local

## Estructura principal

- public/index.php → entrada zona pública
- public/admin.php → entrada zona administración
- app/controllers → controladores
- app/models → modelos
- app/views → vistas
- app/config → configuración
- sql/schema.sql → estructura SQL
